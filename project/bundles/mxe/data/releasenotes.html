Thank you for installing digiKam!


We would like to inform you that port of digiKam under Windows is not as stable as the Linux version.

digiKam must works fine under Windows, but some features are still broken due to some bugs or not yet 
implemented features in the underlying KDE.org libraries, even if since 
digiKam 5.0.0 release, the dependencies to this framework have been seriously reduced.

This installer have be cross-compiled under Linux using MXE project.


License


digiKam is an open-source project published under the General Public Licence version 2 or later.


Internationalization


This installer include a lots of application translations. To change current language,
go to Help menu and use Switch Application Language option. digiKam need to be restarted to take effect.


Documentation


To limit size of this bundle, the large digiKam documentation is not include inside. The complete handbook is available online 
as HTML or 
as PDF.


Requirements and Limitations


Internal database server require to install localy a database backend for Windows, as Mysql or MariaDB.

Camera auto detection do not work yet as under Linux due to limitations in Solid hardware interface used by digiKam.

Panorama and ExpoBlending tools require to install Hugin runtime components to process files.


Contribute and Help


To report a bug or a wish, please follow instructions at digiKam project web site.

To know all current reports open about Windows release, please check the relevant Bugzilla entries.

NOTE: when a dysfunction is discovered please get a backtrace of debug statements generated by digiKam while running.
Not like under Linux, these information are printed on a system console only visible with a dedicated application named DebugView.
You can donwload this tool at this url.
