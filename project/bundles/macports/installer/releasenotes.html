Thank you for installing digiKam!


We would like to inform you that port of digiKam under MacOS is not as stable as the Linux version.

digiKam works mostly fine under MacOS, but some features are still missing due to not yet 
implemented features in the underlying KDE.org libraries, even if since 
digiKam 5.0.0 release, the dependencies to this framework have been seriously reduced. 

This installer has been compiled using the Macports project.


Licence


digiKam is an open-source project published under the General Public Licence version 2 or later.


Internationalization


This installer includes a lot of application translations. To change current language,
go to the Help menu and use the Switch Application Language option. digiKam needs to be restarted for this to take effect.


Documentation


To limit the size of this bundle, the large digiKam documentation is not included. The complete handbook is available online 
as HTML or 
as PDF.


Requirements and Limitations


Internal database server required to install locally a database backend for MacOS, as Mysql or MariaDB.

Camera auto detection does not work yet as under Linux due to limitations in Solid hardware interface used by digiKam.

Panorama and ExpoBlending tools require to install Hugin runtime components to process files.


Contribute and Help


To report a bug or a wish, please follow instructions at digiKam project web site.

To know all current reports open about MacOS release, please check the relevant Bugzilla entries.

NOTE: when a dysfunction is discovered please get a backtrace of debug statements generated by digiKam while running.

Thanks in advance for your support.

The digiKam team
